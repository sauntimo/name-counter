# Name-counter
Node/Typescript app which demonstrates some text parsing

### Installation

- clone the repo
  ```
  $ git clone git@github.com:sauntimo/name-counter.git name-counter
  ```
  
- initialise
  ```
  $ cd name-counter && sudo npm i -g 
  ```

### Usage

```
$ namecounter
```
